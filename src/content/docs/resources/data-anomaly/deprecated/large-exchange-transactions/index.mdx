## Definition

**Large Exchange Transactions** indicates that large transactions have been made.

We take transactions for the last 30 days, calculating thresholds for each asset every day. We take threshold values whose cost exceeds 10,000 USD.
Then we take all transactions whose transaction volume is more than this threshold, separately for each asset.
